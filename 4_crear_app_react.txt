# Crear una apliccación en REACT

1. npx create-react-app sopimuelle         -> crear la aplicación de react
2. cd sopimuelle                           -> ubicarse en la carpeta de la aplicación en react
3. yarn start                              -> ejecutar la aplicación
4. npm install react-bootstrap bootstrap   -> instalar bootstrap en react, ejecutar en la carpeta y de forma global en cmd
5. import 'bootstrap/dist/css/bootstrap.min.css';  -> en cada componente se debe importar el css
6. rfce                                    -> crear la sintaxis basica de un componente en react

7. npm install -g create-react-app         -> instalar componentes con vite
8. function prompt {'React-->'}            -> cambiar de prompt a react
9. npm run build                           -> construir la app para producción por ejemplo apache
10. npm test                               -> Inicia el test de la app
